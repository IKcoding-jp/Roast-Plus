# 要件定義

**Issue**: #350
**作成日**: 2026-03-15
**ラベル**: refactor

## ユーザーストーリー

ユーザー「焙煎室に行くまでの時間の設定はもう使わなくなったので、設定画面から消してほしい」
アプリ「設定項目とおすすめ焙煎時間機能を完全に廃止し、シンプルな設定画面を提供する」

## 要件一覧

### 必須要件
- [ ] 設定画面から「焙煎室に行くまでの時間」入力欄を削除
- [ ] `calculateRecommendedTime`関数を完全削除
- [ ] `goToRoastRoomTimeSeconds`フィールドを型定義から削除
- [ ] 関連するデフォルト値、正規化処理を削除
- [ ] 関連テストの削除・修正

### オプション要件
- なし（完全廃止のため）

## 非機能要件
- 後方互換: 既存LocalStorage/Firestoreに残る古いフィールドが実行時エラーを起こさないこと
- テスト: 既存テストがすべてパスすること

## 受け入れ基準
- [ ] 設定画面に「焙煎室に行くまでの時間」が表示されない
- [ ] おすすめ焙煎時間の提示機能が存在しない
- [ ] TypeScriptコンパイルエラーがない
- [ ] 既存のLocalStorageデータを持つユーザーがエラーにならない
- [ ] `npm run build && npm run test:run` が通る
