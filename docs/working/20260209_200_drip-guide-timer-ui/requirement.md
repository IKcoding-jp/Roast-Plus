# 要件定義

**Issue**: #200
**作成日**: 2026-02-09
**ラベル**: ui, enhancement

## ユーザーストーリー

ユーザー「ドリップ中にスマホをチラ見したとき、今どのステップで次に何をすればいいか一目で分かるようにしてほしい」
アプリ「現在のステップ・次のステップ・残り時間を大きく分かりやすく表示し、ドリップに集中できる」

## 要件一覧

### 必須要件
- [ ] 「今のステップ」が一目で分かる（ステップ名・行動内容）
- [ ] 「次のステップ」が事前に分かる（何をするか・いつ始まるか）
- [ ] 次のステップまでの残り時間が視覚的に表示される
- [ ] 全体の進捗が把握できる（例: 3/5 ステップ）
- [ ] 水量目標が目立つ位置に表示される

### オプション要件
- [ ] ステップ切替時の視覚的アニメーション強化
- [ ] 次のステップへのカウントダウン表示（残り秒数）

## 受け入れ基準

- [ ] スマホで一目で「今何をすべきか」が分かること
- [ ] スマホで一目で「次に何が来るか」が分かること
- [ ] 既存の自動モード・手動モードの動作が壊れていないこと
- [ ] 音声案内機能が正常に動作すること
- [ ] クリスマスモードでも正常に表示されること
- [ ] Lint エラー・warning がゼロであること
