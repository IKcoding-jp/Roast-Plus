# 要件定義

**Issue**: #344
**作成日**: 2026-03-14
**ラベル**: ui

## ユーザーストーリー

ユーザー「iPadの横画面でローストタイマーを開いたら、スマホ用レイアウトが引き伸ばされてて空白だらけ。ちゃんとした表示にしてほしい」
アプリ「768px以上の画面では左にタイマーリング、右にコントロールパネルを配置した左右分割レイアウトで表示する」

## 要件一覧

### 必須要件
- [ ] 768px以上の画面で左右分割型レイアウトに切り替わる
- [ ] 左パネル（50%）にタイマーリング、右パネル（50%）にコントロール
- [ ] リングサイズが左パネル幅の85%（max 340px）に拡大される
- [ ] 3ステート（setup/running/completed）すべてで左右分割が機能する
- [ ] ステート遷移時、右パネルのみフェード/スライドでアニメーション
- [ ] 768px未満のスマホレイアウトは一切変更しない

## 受け入れ基準
- [ ] iPad横画面（1024×768）で左右分割が正しく表示される
- [ ] 3ステートすべてのスクリーンショットが正常
- [ ] スマホ（375×667）で現行レイアウトが維持される
- [ ] 既存テスト全パス（`npm run test:run`）
- [ ] ビルド成功（`npm run build`）
