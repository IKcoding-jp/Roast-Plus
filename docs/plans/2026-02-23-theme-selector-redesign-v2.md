# テーマセレクター リデザイン v2 設計書

**日付**: 2026-02-23
**関連 Issue**: #272 フォローアップ

## 背景

PR #273（feat/#272-theme-selector-redesign）でテーマ設定画面をフルイマーシブデザインにリデザインしたが、以下の未解決課題がある:

1. **アニメーションが控えめすぎる** — 炎の透明度が最大20%など、存在感が薄い
2. **ダーク系テーマの背景が似たり寄ったり** — 7テーマ中5つが非常に暗い単色背景で差別化できていない
3. **LIGHT/DARKバッジがアイコンと重なる** — 同一行配置で視覚的な衝突が発生

## 設計方針

**ハイブリッドアプローチ**: グラデーション背景 + 大型背景アイコン + アニメーション強化を組み合わせ、各テーマに固有の「顔」を作る。

## 1. カードレイアウト変更

### 変更前
```
┌─────────────────────────┐
│ [☕ icon]    [LIGHT badge]│  ← 重なりが発生
│                          │
│ テーマ名 (2xl)            │
│ 説明文 (xs)              │
│                          │
│ [● ● ●]          [✓]   │
└─────────────────────────┘
```

### 変更後
```
┌─────────────────────────┐
│ [☕ icon]                │  ← アイコン単独（左上）
│                  [大型   │
│ テーマ名 (2xl)   アイコン│  ← 背景に大型装飾アイコン（opacity 0.10）
│ 説明文 (xs)     右下配置]│
│                          │
│[LIGHT][● ● ●]    [✓]  │  ← バッジをスウォッチ行に移動
└─────────────────────────┘
```

## 2. 背景グラデーション設計

`lib/theme.ts` の `ThemePreset` に `bgGradient` フィールドを追加し、
`ThemeSelector.tsx` のカードで `backgroundColor: bg` の代わりに使用する。

| テーマ ID | 方向性 | bgGradient 値 |
|-----------|--------|---------------|
| default | 暖かい赤茶 | `135deg, #261a14 0%, #3d2008 100%` |
| dark-roast | 赤熱の深淵 | `135deg, #0d0b09 0%, #200a04 100%` |
| light-roast | 朝の光差し込み | `135deg, #faf6ef 0%, #ede0c8 100%` |
| matcha | 深い竹林 | `135deg, #0f1f15 0%, #041a09 100%` |
| caramel | 焦げキャラメル | `135deg, #1a120d 0%, #2e1200 100%` |
| christmas | 深緑から夜空 | `135deg, #051a0e 0%, #100520 100%` |
| dark | 宇宙の深淵 | `135deg, #0f0f0f 0%, #0f1020 100%` |

## 3. アニメーション強化値

| 種類 | 変更前 | 変更後 | 変更内容 |
|------|--------|--------|---------|
| steam 透明度 | `[0, 0.35, 0]` | `[0, 0.65, 0]` | +30pt |
| flame 透明度 | `[0.08, 0.2, 0.08]` | `[0.3, 0.65, 0.3]` | 約3倍 |
| flame スケール | `[1, 1.1, 1]` | `[1, 1.35, 1]` | より大きく揺らぐ |
| leaf 透明度 | `0.12`（固定） | `[0.15, 0.35, 0.15]`（アニメ） | 動的に変化 |
| leaf 回転 | `[-4, 4, -4]deg` | `[-10, 10, -10]deg` | 2.5倍の揺れ |
| glow 色強度 | `${color}22`（13%） | `${color}55`（33%） | 2.5倍 |
| snow 透明度 | `0.65` | `0.9` | より鮮明な雪 |
| stars 透明度 | `[0.8, 0.1, 0.8]` | `[1, 0.05, 1]` | よりくっきり点滅 |

## 4. 大型背景アイコン

各カードの右下に装飾用の大型アイコンを追加（コンテンツとは別の背景層）:

```tsx
{/* 大型背景アイコン装飾 */}
<div className="absolute bottom-0 right-1 pointer-events-none">
  <Icon size={90} style={{ color: accent, opacity: 0.10 }} aria-hidden />
</div>
```

---

## 5. バッジ位置変更

下部バーを再設計し、LIGHT/DARKバッジをスウォッチの左側に配置:

```tsx
{/* 下部: バッジ + 色スウォッチ + 選択チェック */}
<div className="flex items-center gap-2">
  {/* LIGHT/DARKバッジ */}
  <span ...>{preset.type === 'light' ? 'LIGHT' : 'DARK'}</span>
  {/* 色スウォッチ */}
  <div className="flex gap-1.5">...</div>
  {/* 右端にチェック */}
  <div className="ml-auto">...</div>
</div>
```
