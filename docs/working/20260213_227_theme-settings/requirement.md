# 要件定義

**Issue**: #227
**作成日**: 2026-02-13
**ラベル**: enhancement

## ユーザーストーリー

ユーザー「毎日使うアプリだから、自分好みの見た目にカスタマイズしたい。ダークモードも欲しいし、コーヒーらしいテーマがいくつかあると楽しい」
アプリ「設定画面のテーマ設定セクションから、6つのテーマプリセットをワンタップで切り替えられます。選んだテーマは全ページに即座に反映されます」

## 要件一覧

### 必須要件

- [ ] 6つのテーマプリセットの実装（デフォルト、ダークロースト、ライトロースト、抹茶ラテ、キャラメルマキアート、クリスマス）
- [ ] 設定画面に「テーマ設定」セクションを追加（カードグリッドUI）
- [ ] テーマ切替が全ページに即時反映される
- [ ] クリスマスモードをテーマの1つとして統合（既存Switchを廃止）
- [ ] テーマ選択がlocalStorageに保存され、再訪問時に復元される
- [ ] 各テーマのCSS変数セットが全共通UIコンポーネントに適用される

### オプション要件

- [ ] テーマプレビューカードにミニチュアUIプレビューを表示
- [ ] テーマ切替時のスムーズなトランジションアニメーション

## テーマプリセット仕様

| テーマID | テーマ名 | タイプ | コンセプト |
|---------|---------|--------|-----------|
| `default` | デフォルト | ライト | 既存。暖かいコーヒー系ライトテーマ |
| `dark-roast` | ダークロースト | ダーク | 深煎りエスプレッソ。漆黒+琥珀&ゴールドの高級感 |
| `light-roast` | ライトロースト | ライト | 浅煎りの明るさ。クリーム〜アイボリー+ハニーイエロー |
| `matcha` | 抹茶ラテ | ダーク | 和カフェの落ち着き。抹茶グリーン+クリームホワイト |
| `caramel` | キャラメルマキアート | ダーク | 秋の収穫祭。チョコレートブラウン+キャラメルゴールド |
| `christmas` | クリスマス | ダーク | 既存。深緑+ゴールド+装飾エフェクト |

## 受け入れ基準

- [ ] 設定画面でテーマカードをタップすると即座にテーマが切り替わる
- [ ] 全ページ（ホーム、各機能ページ、設定）でテーマが正しく反映される
- [ ] ブラウザリロード後もテーマが維持される
- [ ] クリスマステーマ選択時に既存の装飾（snowfall、アイコン置き換え）が動作する
- [ ] 既存の `useChristmasMode` hook 利用箇所が破壊されない（後方互換性）
- [ ] 共通UIコンポーネント（Button, Card, Input等）が全テーマで正しく表示される
- [ ] Lintエラー・warning ゼロ
