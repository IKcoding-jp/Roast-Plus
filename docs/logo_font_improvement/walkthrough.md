# ロゴフォントの改善：ウォークスルー

ホーム画面、スプラッシュ画面、ログイン画面のロゴを、コーヒー専門店のような上品で洗練されたデザインにアップデートしました。

## 実施した変更

### タイポグラフィの刷新
- **英語部分 (`Roast Plus`)**: 高級感と伝統を感じさせるセリフ体 `Playfair Display` を採用しました。
- **日本語部分 (`ローストプラス`)**: 洗練された明朝体 `Noto Serif JP` を使用し、文字間隔（レタースペーシング）を広めに取ることで、ゆとりのある上品な印象を演出しました。

### デザインの統一と配色の微調整
アプリ全体のブランディングを強化し、視認性を高めるために以下の調整を行いました。

1. **ホーム画面**: ダークヘッダーに合わせて `Roast` 部分をホワイトに変更。視認性と高級感を両立させました。
2. **スプラッシュ画面**: 背景を深みのあるダークブラウン (`#1a1412`) に変更し、中央の白いロゴが美しく浮かび上がるように演出。
3. **ログイン画面**: ロゴエリアに専用のダーク背景セクションを設け、一貫した世界観を提供。

## 変更内容の確認

### ホーム画面のロゴ
ダークなヘッダーに白い `Roast` とオレンジの `Plus` が鮮やかに映えるデザインになりました。
クリスマスモード時は、既存の華やかなアニメーション付きロゴが維持されるよう配慮しています。

### スプラッシュ画面
新しいダークテーマにより、アプリ起動時の第一印象がよりドラマチックでプレミアムなものになりました。

### ログイン画面
中央のロゴセクションがアイコニックに強調され、プロフェッショナルなツールとしての風格を醸し出しています。

---

以上の変更により、視認性を確保しつつ、アプリ全体が「こだわりのコーヒーを扱うプロフェッショナルな空間」としての質感を高めることができました。
