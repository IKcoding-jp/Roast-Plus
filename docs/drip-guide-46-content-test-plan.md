## 目的

4:6メソッド（粕谷）の **「文言改善」** が、開始前と抽出中に正しく表示され、既存のドリップガイド体験を壊さないことを確認する。

---

## スモークテスト（最短）

- [ ] `/drip-guide` を開く
- [ ] 「4:6メソッド（粕谷）」→「ガイド開始」で `Start46Dialog` が開く
- [ ] `Start46Dialog` に「落ち切り優先／時刻は目安」が見える（折りたたみ内でもOKだが、**迷わない視認性**があること）
- [ ] 任意の条件で開始し `/drip-guide/run` を開く
- [ ] Runnerの各ステップで、ヒント枠（電球）に「落ち切り優先（時刻は目安）」が表示される

---

## 表示内容（開始前）

`Start46Dialog` の解説ブロックを開いて確認:

- [ ] 4:6のコア（40%/60%の考え方）が説明されている
- [ ] 「粉量×15（例：20g→300g）」が説明されている
- [ ] **最重要**：「次は“ほとんど落ち切ってから”。時刻は目安」が明確
- [ ] 準備条件（水/湯温/挽き目）の案内がある（初回失敗を減らす）
- [ ] 文言が“公式文のコピペ”になっていない（言い換え・要約になっている）

---

## 表示内容（抽出中）

### 共通ヒント（全ステップ）

- [ ] 3投/4投/5投のどの場合でも、**全ステップ** のヒント枠に共通文言が出る
- [ ] 共通ヒントの内容が「落ち切り優先／時刻は目安」になっている

### ステップ説明（description）

少なくとも以下を目視確認:

- [ ] 1投目（蒸らし）の説明が「粉全体を均一に…」など、目的が分かる短文になっている
- [ ] 2投目が「味の40%（味わいの方向性）」に触れている
- [ ] 3投目以降が「濃度の60%（濃度調整）」に触れている
