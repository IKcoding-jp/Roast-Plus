# 要件定義

**Issue**: #236
**作成日**: 2026-02-13
**ラベル**: bug

## ユーザーストーリー
ユーザー「ページを切り替えると一瞬白くチカッとして目に悪い。特にダーク系テーマだと顕著」
アプリ「ページ遷移時にフラッシュなく、スムーズに背景色が維持される」

## 要件一覧

### 必須要件
- [ ] ページ遷移時に白フラッシュが発生しないこと
- [ ] 全テーマ（default, christmas, dark-roast, light-roast, matcha, caramel）で検証済み
- [ ] Loading コンポーネントがテーマに追従すること
- [ ] 既存のテーマ切替機能に影響しないこと

### オプション要件
- [ ] `--background` 変数の整理（`--page` への統合検討）

## 受け入れ基準
- [ ] デフォルトテーマでページ遷移時に白フラッシュが発生しない
- [ ] クリスマスモード等ダーク系テーマで遷移時に明るいフラッシュが発生しない
- [ ] Loading表示がテーマカラーと一致する
- [ ] ビルドエラー・Lintエラーがゼロ
- [ ] 既存テストが全件パス
