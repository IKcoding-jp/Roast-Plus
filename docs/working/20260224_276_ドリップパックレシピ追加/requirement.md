# requirement.md — #276 ドリップパックレシピ追加

## 概要

ドリップパック（7〜10g）用デフォルトレシピをドリップガイドに追加する。
同時に、手動モードレシピの「タップで手順を進める」操作説明が欠けているUX問題を修正する。

## 背景・理由

- 職場でドリップパックを使うメンバーがいるが、対応レシピが存在しない
- 既存の BYSN Standard / 井崎流と同じ `isManualMode: true` で統一感を持たせる
- 手動モードで操作説明なし → 初回ユーザーが「タップで次へ進む」ことを知れない

## レシピ仕様

### 基本パラメータ

| フィールド | 値 |
|-----------|---|
| id | `recipe-drip-bag` |
| name | `ドリップパック（7〜10g）` |
| beanName | `BYSNドリップパック` |
| beanAmountGram | `7`（表示上は「7〜10g」と説明に記載） |
| totalWaterGram | `155`（仕上がり140ml＋粉吸収分） |
| totalDurationSec | `150`（2分30秒） |
| isDefault | `true` |
| isManualMode | `true` |

### ステップ定義

| step id | title | targetTotalWater | note |
|---------|-------|:----------------:|------|
| step-0 | カップ温め | — | お湯を注いで30秒待ち、捨てる |
| step-1 | パックのセット | — | 開封前に振って粉を均す→カップにセット |
| step-2 | 蒸らし（1投目） | 30 | 25〜30ml。1分待つ。2投目は落ちきる前に注ぎ始める |
| step-3 | 2投目 | 93 | 60〜65ml。30秒かけて中心から細く注ぐ |
| step-4 | 3投目 | 155 | 60〜65ml。表面が沈んだら30秒かけて注ぐ |
| step-5 | 引き上げ | — | 2:10〜2:30を目安に取り出す。つけ置きしない |

### 注意事項（description に記載）

- 一気に注がない、細く注ぐ
- お湯の量が多すぎると薄くなる（155mlを守る）
- 引き上げが遅いと雑味が出る

## 手動モード UI 改善仕様

`StartHintDialog` に `isManualMode?: boolean` prop を追加する。

- `isManualMode: true` の場合、既存の2説明に加えて「手順はタップで進みます」セクションを表示
- 内容: 「ガイドの各手順は画面下の「次へ」ボタンをタップして手動で進めます。タイマーは経過時間の目安として動きます。」

## 受け入れ基準

- [ ] レシピ一覧に「ドリップパック（7〜10g）」が表示される
- [ ] デフォルト表示順: BYSN → 井崎 → 粕谷 → Hoffmann → **ドリップパック**
- [ ] ガイド開始前ダイアログに「手順はタップで進みます」説明が表示される（isManualMode時）
- [ ] ランナー画面で6ステップが順番に表示され、手動で進められる
- [ ] lint / build / test すべてパス
