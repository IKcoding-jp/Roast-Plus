# 実装計画：ホーム画面ドリップガイドへの「新レシピ登場！」ラベル追加

ホーム画面の「ドリップガイド」カードに、ユーザーの目を引く「新レシピ登場！」ラベルを追加します。既存の `new-label-gradient` スタイルを活用し、プレミアムな外観を実現します。

## 変更内容

### 1. `app/page.tsx` の修正
- `ACTIONS` 配列のの「ドリップガイド」項目に `badge` プロパティを追加します。
- カードのレンダリングループ内で、`badge` が存在する場合にラベルを表示する処理を追加します。
- ラベルには `globals.css` で定義されている `new-label-gradient` と `animate-pulse-scale` を適用し、高級感のあるアニメーションを実現します。

## 影響範囲
- ホーム画面のカードレイアウト。
- 「ドリップガイド」カードのみにラベルが表示されます。

## 検証計画
- ブラウザでホーム画面を表示し、「ドリップガイド」カードの左上に「新レシピ登場！」ラベルが表示されていることを確認します。
- ラベルがグラデーションアニメーションおよびパルスアニメーションをしていることを確認します。
- モバイル表示でもレイアウトが崩れていないことを確認します。
