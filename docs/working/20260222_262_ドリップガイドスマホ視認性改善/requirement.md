# requirement.md — Issue #262

## 概要
ドリップガイド実行画面（`/drip-guide/run`）のスマホ視認性を改善する。
タイマー数字と注水量の文字サイズを拡大し、ハンドドリップ中でも一瞬で読み取れるようにする。

## 背景
- 8名のチームが毎日使う現場ツール
- コーヒーを注ぎながら片手で画面を確認するユースケース
- 現在のタイマー 4.5rem / 注水量 2.25rem は小さい
- Design Lab で A/B/C 3案を比較し、**Option A（タイマー・注水量のみ拡大）** を採用

## 採用案（Option A）の変更内容

| 要素 | 変更前 | 変更後 |
|------|--------|--------|
| タイマー数字 | `text-[4.5rem]` (72px) | `text-[5.5rem]` (88px) |
| 注水量数字 | `text-[2.25rem]` (36px) | `text-[3rem]` (48px) |
| 注水量単位 g | `text-lg` (18px) | `text-xl` (20px) |

## スコープ（含まない）
- ステップタイトル・説明文・カウントダウン秒数のサイズ変更 → 不要（Option B を不採用）
- ボトムボタンのサイズ変更 → 変更なし
- 自動モード・手動モードの振る舞い変更 → 変更なし

## 受け入れ基準
- [ ] タイマー数字が 5.5rem で表示される
- [ ] 注水量数字が 3rem で表示される
- [ ] ボトムボタンサイズが変化していない
- [ ] lint / build / test がすべてパス
