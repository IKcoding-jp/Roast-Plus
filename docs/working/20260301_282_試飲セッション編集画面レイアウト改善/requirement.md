# 要件定義

**Issue**: #282
**作成日**: 2026-03-01
**ラベル**: ui

## ユーザーストーリー

ユーザー「試飲セッションを編集しようとしたら、フォームが上に寄っていて余白が多い。ボタンも3つあって1つが改行されていて使いにくい」
アプリ「編集フォームを画面縦中央に表示し、削除はアイコンボタン化して、キャンセル＋更新の2ボタンを1行に整列させる」

## 要件一覧

### 必須要件
- [ ] セッション編集フォームが画面縦中央に表示される
- [ ] 削除ボタンをフォーム右上の小さいアイコンボタン（ゴミ箱）に変更する
- [ ] ボトムボタン行は「キャンセル」「更新する」の2ボタンのみ、常に横1行で表示される（改行なし）

## 受け入れ基準
- [ ] 編集フォームが画面の縦中央に配置されている
- [ ] ゴミ箱アイコンボタンがフォーム右上に表示されている（既存のDeleteダイアログが動作すること）
- [ ] ボタン行が狭い幅でも「キャンセル」「更新する」の2つが1行に収まっている
- [ ] 新規作成時（`isNew=true`）はゴミ箱ボタンが表示されないこと
