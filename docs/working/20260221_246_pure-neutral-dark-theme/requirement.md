# requirement.md — Pure Neutral Dark テーマ追加

**Issue**: #246
**日付**: 2026-02-21
**タイプ**: feat

---

## 目的

目の疲れ防止・ブルーライト軽減のための実用的ダークテーマを7番目のテーマとして追加する。

## ユーザーストーリー

> 「夜間や長時間使用時に目が疲れにくいシンプルなダークモードを使いたい。
> コーヒーをテーマにしたダークテーマではなく、VS Code や GitHub のような汎用ダークモードがほしい。」

## 受け入れ基準

- [ ] `data-theme="dark"` で Pure Neutral Dark テーマが適用される
- [ ] `/settings/theme` のテーマセレクターに「ダークモード」として表示される
- [ ] 全 CSS セマンティックトークン（--page, --surface, --ink 等）が定義されている
- [ ] アクセントカラーは既存オレンジ `#d97706` を維持している
- [ ] テキストコントラスト比 WCAG AA 基準（4.5:1）以上を満たす（実際: 約14.5:1）
- [ ] ThemeProvider のテストが通過する
- [ ] lint / build / test がすべて通過する

## スコープ外

- OS のダークモード設定への自動連動（別 Issue として扱う）
- 既存テーマの変更
- テーマ切り替えアニメーションの変更

## テーマ仕様

| 属性 | 値 |
|------|-----|
| `id` | `'dark'` |
| `name` | `'ダークモード'` |
| `description` | `'目の疲れを抑える汎用ダークテーマ'` |
| `type` | `'dark'` |
| `themeColor` | `'#0f0f0f'` |
