# 要件定義

**Issue**: #201
**作成日**: 2026-02-09
**ラベル**: ui

## ユーザーストーリー

ユーザー「試飲記録ページの見た目が他のページと違って統一感がない」
アプリ「共通コンポーネントを使った枠線のあるカード表示で、スクロールも適切に動作する」

## 要件一覧

### 必須要件
- [ ] 空状態表示を `<Card>` コンポーネントに置き換え（`motion.div` から移行）
- [ ] 枠線のあるカードの見た目で統一
- [ ] モバイルのスクロールバー位置を見直し（`h-[calc(100vh-140px)]` の修正）
- [ ] `pb-20 sm:pb-0` の下部余白問題を修正

### オプション要件
- [ ] `overflow-y-hidden` の見直し（デスクトップでのスクロール制御）

## 受け入れ基準

- [ ] 空状態表示が `<Card>` コンポーネントで表示されること
- [ ] カードに枠線（`border border-edge`）が表示されること
- [ ] モバイルでスクロールが正常に動作すること（不要な余白なし）
- [ ] デスクトップでも適切にスクロールできること
- [ ] 既存の機能（セッション作成、記録入力、フィルタ等）が壊れていないこと
- [ ] クリスマスモードでも正常に表示されること
- [ ] Lint エラー・warning がゼロであること
