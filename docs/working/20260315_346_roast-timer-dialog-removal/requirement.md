# 要件定義

**Issue**: #346
**作成日**: 2026-03-15
**ラベル**: refactor

## ユーザーストーリー

ユーザー「焙煎が終わったら、すぐに完了画面を見たい。毎回ダイアログをOKする手間が面倒」
アプリ「焙煎完了時にダイアログを表示せず、直接完了画面（統計＋リセットボタン）を表示する」

## 要件一覧

### 必須要件
- [ ] 焙煎完了時にCompletionDialog（「もうすぐ焙煎が完了します。」）を表示しない
- [ ] ContinuousRoastDialog（「続けて焙煎しますか？」）を表示しない
- [ ] AfterPurgeDialog（「お疲れ様でした！」）を表示しない
- [ ] タイマー完了時は既存の完了画面（緑リング＋統計カード＋リセットボタン）を直接表示
- [ ] ダイアログ関連のコード・型・フックを完全削除（デッドコードなし）
- [ ] Firestore既存データとの後方互換性を維持

## 受け入れ基準
- [ ] `npm run build` が成功する
- [ ] `npm run test:run` が全テスト合格
- [ ] `npm run deadcode` でダイアログ関連のデッドコードが検出されない
- [ ] タイマー完了時にダイアログが表示されず、完了画面が直接表示される
