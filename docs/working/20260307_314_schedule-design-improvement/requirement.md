# 要件定義

**Issue**: #314
**作成日**: 2026-03-07
**ラベル**: ui

## ユーザーストーリー

ユーザー「スケジュールページが古いLLMで作ったままで地味。担当表みたいな洗練されたデザインにしてほしい」
アプリ「統一感のあるスタイリング、スムーズなタブ切替アニメーション、分かりやすい空状態ガイドを提供する」

## 要件一覧

### 必須要件
- [ ] 日付ナビゲーションのスタイリング改善（ボーダー・シャドウ・タイポグラフィ）
- [ ] 下部タブナビをセグメントコントロール風に改善（Framer Motionインジケーター）
- [ ] TodaySchedule/RoastSchedulerTabのカードスタイリング改善
- [ ] ScheduleCardのデザイン改善（角丸・カラーバー・ホバー効果）
- [ ] 空状態のガイドカード化（Card variant="guide"使用）

### 制約
- 機能・データ構造は変更しない
- CSS/JSXスタイリング + Framer Motionアニメーションのみ
- テーマ対応: CSS変数ベースを維持（ハードコード色をCSS変数に置換）

## 非機能要件
- モバイル対応: 既存のレスポンシブ構造を維持
- アクセシビリティ: 既存のaria-label等を維持

## 受け入れ基準
- [ ] 7テーマすべてで表示崩れがないこと
- [ ] モバイル/タブレット/デスクトップで正常表示
- [ ] 既存の機能（OCR、日付ナビ、タブ切替、ドラッグ並替え等）が動作すること
- [ ] lint / build / test がすべてパスすること
