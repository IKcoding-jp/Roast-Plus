# 要件定義

**Issue**: #321
**作成日**: 2026-03-08
**ラベル**: ui

## ユーザーストーリー

ユーザー「Modalの開閉アニメーションが派手すぎて好みじゃない。もっと洗練されたものにしたい」
アプリ「デザインラボで複数パターンを比較し、選んだアニメーションで統一的に動作する」

## 要件一覧

### 必須要件
- [ ] デザインラボに「Modalアニメーション」比較セクションを追加
- [ ] 5パターンのアニメーション候補を実際に開閉して比較できる
- [ ] 選定したパターンをModal.tsxに適用

### オプション要件
- [ ] 追加パターンの候補（ユーザーフィードバックに応じて）

## 受け入れ基準
- [ ] デザインラボで5パターンのアニメーションを比較できる
- [ ] 選定パターンがModal.tsxに適用され、Dialog含む全モーダルに反映される
- [ ] lint / build / test がすべてパス
