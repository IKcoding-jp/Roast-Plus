# 要件定義: テーマ設定画面パターンBデザイン刷新

## Issue
- **Issue番号**: #338
- **タイプ**: style
- **規模**: 中

## 背景
現在のテーマ設定画面は以下の問題を抱えている:
- 7カード中5枚がダーク系グラデーション背景で全体が暗い
- 7種のFramer Motionアニメーション（Steam/Flame/Particles/Leaf/Glow/Snow/Stars）が常時ループし視覚的ノイズが多い
- グラデーション背景 + 90px装飾アイコン + アニメーション層 + コンテンツの4層が競合しごちゃごちゃしている

## 要件
デザインプレビュー（`design-preview.html`）で確定したパターンB（カラードット）に刷新する。

### 確定デザイン仕様

**カード構造:**
- 白背景 (`bg-surface`)
- ボーダー: 1.5px `border-edge-subtle`
- 角丸: 16px (`rounded-2xl`)
- パディング: 20px上 14px左右 16px下
- 中央揃え、縦方向レイアウト

**カラードット:**
- サイズ: 48px 丸
- グラデーション: テーマごとの2色グラデーション（中間明度）
- 内側リング: 3px inset、白50%透明度

**テキスト:**
- テーマ名: 15px / font-weight 700 / text-ink
- 説明文: 12.5px / #6b7280 (text-ink-sub)

**選択状態:**
- ボーダー: 2px `border-spot` (#d97706)
- リング: 3px rgba(217,119,6,0.12)
- 右上にチェックマーク: 20px丸、#d97706背景、白チェック

**グリッド:**
- デスクトップ (640px+): 4列、gap 12px
- モバイル: 2列、gap 10px
- max-width: 960px

**ドットのグラデーション色:**
| テーマ | From | To |
|--------|------|-----|
| デフォルト | #8b6530 | #c9a06a |
| ダークロースト | #3d2815 | #7a5530 |
| ライトロースト | #f5e8cc | #e5cf98 |
| 抹茶ラテ | #2d5a3a | #5a9a60 |
| キャラメルマキアート | #8a5520 | #c88a40 |
| クリスマス | #1a4028 | #356a42 |
| ダークモード | #252530 | #454560 |

### 削除するもの
- 全Framer Motionアニメーション（7種）
- `ThemeAnimationType` 型
- `bgGradient` プロパティ
- `fontStyle` プロパティ
- `animationType` プロパティ
- 90px装飾アイコン（カード背景）
- LIGHT/DARKバッジ
- 3色スウォッチ

## 受け入れ基準
1. テーマ設定画面がパターンBデザインで表示される
2. 7テーマすべてが正しいグラデーション色のドットで表示される
3. テーマ選択が正しく動作する（クリックでテーマ切り替え）
4. 選択中テーマにチェックマーク（右上）が表示される
5. レスポンシブ: デスクトップ4列/モバイル2列
6. アニメーションが完全に削除されている
7. lint / build / test が通る
8. 既存テストが新構造に更新されている
