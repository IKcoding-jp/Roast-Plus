# 要件定義

**Issue**: #194
**作成日**: 2026-02-08
**ラベル**: bug

## ユーザーストーリー
ユーザー「クリスマスモードの雪がカードの上にかぶさって見づらい」
アプリ「雪の演出はカードの背面に表示し、カードの視認性を確保する」

## 要件一覧
### 必須要件
- [ ] Snowfallコンテナのz-indexをカードより低い値に変更
- [ ] 雪片がカード要素の下に表示されることを確認

### オプション要件
- なし

## 受け入れ基準
- [ ] クリスマスモードで雪がカードの背面に表示される
- [ ] カードのテキスト・アイコンが雪に遮られない
- [ ] 雪の演出自体は引き続き表示される（完全に見えなくならない）
