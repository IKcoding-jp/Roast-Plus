# 要件定義

**Issue**: #229
**作成日**: 2026-02-13
**ラベル**: ui

## ユーザーストーリー
ユーザー「テーマを抹茶ラテに変えたのに、ステータスバーとナビゲーションバーが前の色のままだ」
アプリ「テーマ変更時に、ステータスバー・ナビゲーションバーの色もテーマに合わせて自動変更する」

## 要件一覧
### 必須要件
- [ ] テーマ変更時に `<meta name="theme-color">` を動的に更新する
- [ ] 全6テーマに適切な `themeColor` 値を定義する
- [ ] PWAインストール済み環境でステータスバー色がテーマに連動する
- [ ] 3ボタンナビゲーションバーの背景がテーマカラーに連動する

### オプション要件
- [ ] `site.webmanifest` の `background_color` をデフォルトテーマの `--page` に合わせる

## 受け入れ基準
- [ ] テーマ変更後、ステータスバー（バッテリー・WiFi表示エリア）の色がテーマに合った色に変わる
- [ ] 3ボタンナビゲーションバーの背景がテーマカラーで表示される
- [ ] ページリロード後もテーマカラーが維持される
- [ ] デフォルトテーマでの初回表示が従来と変わらない
