# クリスマス仕様ホーム画面の実装完了

ホーム画面を伝統的な「王道クリスマス」デザインに変更し、設定画面からオンオフを切り替えられる機能を実装しました。

## 変更内容

### 1. 状態管理の導入
- [useChristmasMode.ts](file:///d:/Dev/roastplus/hooks/useChristmasMode.ts) を作成。
- `localStorage` を使用して設定を永続化し、デフォルトで「オン」になるようにしました。

### 2. 設定画面の更新
- [app/settings/page.tsx](file:///d:/Dev/roastplus/app/settings/page.tsx) に「クリスマスモード」のトグルスイッチを追加しました。
- 直感的にわかるよう、🎄アイコンを添えています。

### 3. 雪のアニメーション
- [components/Snowfall.tsx](file:///d:/Dev/roastplus/components/Snowfall.tsx) を作成。
- CSSアニメーションを利用した軽量な雪の粒子を、ホーム画面の背景に降らせています。

### 4. ホーム画面のデザイン変更
- [app/page.tsx](file:///d:/Dev/roastplus/app/page.tsx) を大幅にアップデート。
- **配色**: 背景を深い緑のグラデーションに、ヘッダーをクリスマスレッドに変更。
- **アイコン**: 各機能を象徴するクリスマスアイコン（ギフト、ツリー、ベル、ジンジャーブレッドマンなど）を動的に適用。
- **スタイル**: カードのボーダーにゴールド（真鍮色）の装飾を加えました。

## 検証結果

- [x] 設定画面でトグルをオフにすると、即座にデフォルトのコーヒー色のデザインに戻る。
- [x] 設定画面でトグルをオンにすると、華やかなクリスマスデザインと雪のアニメーションが表示される。
- [x] ページをリロードしても設定が維持される。
- [x] `GiCandyCanes` などのアイコンライブラリの型エラーを修正済み。
