# 要件定義

**Issue**: #342
**作成日**: 2026-03-14
**ラベル**: ui

## ユーザーストーリー

ユーザー「焙煎機に豆をセットした。200gだから8分後に1階の焙煎室に戻りたい。ワンタップでタイマーをスタートしたい」
アプリ「重さカード（200g/300g/500g）をタップ → スタートで即座にカウントダウン開始。完了時に通知。」

## 要件一覧

### 必須要件
- [ ] 1画面で idle → running → completed が遷移（画面切替なし）
- [ ] 重量カード3択: 200g→8分, 300g→9分, 500g→10分
- [ ] リング（円形プログレス）が全ステートで同じ位置に表示
- [ ] アクションボタンが全ステートで同じ縦位置
- [ ] Fraunces セリフ体フォントで時間表示
- [ ] 60本ティックマーク付きリングデザイン
- [ ] 全7テーマ対応（セマンティックCSS変数のみ使用）
- [ ] FloatingNavパターンのヘッダー

### 不要な機能（削除対象）
- おすすめタイマー画面
- 手動時間入力
- 記録一覧へのリンク

## 非機能要件
- パフォーマンス: rAFベースの60fpsアニメーション維持
- アクセシビリティ: 44pxタッチターゲット、aria-label
- モバイル対応: 375px〜対応、レスポンシブリング

## 受け入れ基準
- [ ] 重量カードタップ → リング時間更新 → スタートの一連フロー
- [ ] 全3ステートでリングの縦位置が変わらない
- [ ] 全3ステートでボタンの縦位置が変わらない
- [ ] 7テーマすべてで視覚的に破綻なし
- [ ] `npm run build && npm run test:run` 通過
