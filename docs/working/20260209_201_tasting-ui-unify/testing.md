# テスト計画

## テスト戦略

### 手動テスト（メイン）

UI統一作業のため、目視確認が中心。

#### カード表示確認
- [ ] 空状態のカードに枠線が表示されること
- [ ] 空状態のアニメーション（motion）が正常に動作すること
- [ ] 他のページ（担当表、ドリップガイド）とカードの見た目が統一されていること

#### スクロール確認（モバイル）
- [ ] 下部に不要な余白がないこと
- [ ] 横スクロール（カルーセル）が正常に動作すること
- [ ] ページインジケーターが正しく表示されること

#### スクロール確認（デスクトップ）
- [ ] 縦スクロールが正常に動作すること
- [ ] コンテンツが途切れないこと

#### 機能確認
- [ ] セッション一覧の表示が正常
- [ ] セッション作成が正常
- [ ] 記録入力が正常
- [ ] フィルター機能が正常

#### テーマ確認
- [ ] 通常モードで正常表示
- [ ] クリスマスモードで正常表示

### 自動テスト

- [ ] 既存テストの pass 確認
- [ ] Lint チェック pass

## カバレッジ目標

- 既存テストの維持（新規テスト追加は不要、UIの見た目変更のため）
